First click — browser autoplay policy means audio won't start until you interact. Click Play after launching.
Drop audio — drag WAV, MP3, OGG, or FLAC files from your file manager directly onto any track lane to import them.
Drum Machine — open it from the topbar Plugins button. Hit Bounce to drop a rendered loop onto a new track.
Save session — use ◈ Save to name and store your session. Sessions survive browser refresh and are stored in IndexedDB.
Export .wavrproj — use the export button to create a portable project file you can share or open on another machine.
Templates — click ⊕ New to open the template picker. Six synthesized starters (lo-fi, techno, ambient, pop, podcast, blank) render instantly.
EQ — click the EQ button on any track header to open the 6-band equalizer. Changes apply live during playback.
